Popoludnie s jazykom Java 2007

2007/02/19

O projekte

Popoludnie s jazykom Java je neformálny ,,hands-on" seminár venujúci sa jazyku Java a súvisiacim technológiám pod vedením Róberta Novotného Prebieha nepravidelne v piatky od 13.00 v laboratóriu P7.

V súčasnosti beží už druhý ročník - tentokrát v pozmenenej podobe.

Program

Servlety a JSP

  • Inštalácia servera Apache Tomcat 5.5.x na Windowse
  • Vytvorenie a nasadenie webových aplikácií pre Tomcat
  • Servlety a JSP - bakalárska práca
  • Slajdy o servletoch a JSP
  • Miniseriál S Javou na webovém serveru na Root.cz. Diel 1, Diel 2, Diel 3
  • Core Servlets and JSP. Voľne stiahnuteľná príručka k servletom a JSP (staršieho dáta, ale mnoho vecí platí)
  • Java Server Pages - obsiahly seriál na Interval.cz
  • Java Servlets - ďalší obsiahly seriál na Interval.cz
